Air CSS ver. 1.0.0

Air is a minimalist class-based CSS framework leveraging commutativity and composition features to rapidly build anything from simple web pages to a full-featured webapp. All of this directly in your HTML.

Air is utterly influenced by Tachyons and started its life as a fork of it.

Features

Classes provided by air have a very low specificity and are used to set predefined values to carefully chosen CSS properties. No class should override your existing styles. Each class belongs to a family of properties among the following: typography, layout and theming.

The name of a class is built around a base indicating the property it modifies and, if necessary, a modifier indicating the value associated to this property. There is some exceptions to this rule for the most common property and value combinations with the use of short-handed class names.

Air is mobile-first and offers the possibility to target larger breakpoints with the help of namespaced classes in typography and layout property families.

A ready-to-use color palette is provided with air along with some common filters. Most of them support a hovering mode with smooth transitions.
