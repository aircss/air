Air CSS ver. 1.0.0

Font style

Single purpose class to display a piece of content in italic.

Mnemonic

There is no mnemonic for font family.

Shorthands

There is one shorthand to style the text content of a page or an element:

  • i is used to display an italic font style

Examples

<h1 class="i f-title lh-solid">Title</h1>
