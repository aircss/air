Text decoration

Underline and strikethrough decorations on a text are respectively used to add an emphasis or epanorthosis to all or part of the latter.

Mnemonic

Base
Modifiers
Optional extensions
strike
underline
none
none

Examples

<h1>We are feeling <span class="strike">terrible</span> fine.</h1>

We are feeling terrible fine.

<h1>This information is <span class="underline">important</span>.</h1>

This information is important.
