Coordinates

The top, right, bottom, and left classes determine the final location of positioned elements inside of a relative element and to a lesser extent in a flexbox.

Mnemonics

Bases
Modifiers
Optional extensions
top
right
bottom
left
none
-m = medium width container
-l = large width container

Examples

.top
.right
.bottom
.left
.top .left
.top .right
.bottom .right
.bottom .left
no coordinates
<div class="relative pa7">
  <div class="absolute top">.top</div>
  <div class="absolute bottom left">.bottom .left</div>
  …
</div>

This example shows how absolute elements are absolutely positioned inside of a relative element. The absolute element outer margin snapping to the containing element is highlighted in each case.
